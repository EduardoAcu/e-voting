Acerca de Nosotros


Este proyecto tiene como objetivo desarrollar y diseñar un sistema de votación en línea rápido, fácil y seguro.


Este proyecto fue realizado por estudiantes de 4to año de la carrera de Ingenieria en Informatica, que actualmente están matriculados en el Instituto Profesional Inacap.

Servicios

Rápido

Este sistema de votación nos ayudará a minimizar la duración del tiempo de votación porque además de proporcionar a los votantes una plataforma en línea para votar, el sistema contará automáticamente los votos de cada candidato.

Fácil

Los desarrolladores de este sistema crearon una interfaz de usuario sencilla para que esta aplicación web sea fácil de usar.

Seguro

Este sistema está hecho por Django, un framework de Python con una seguridad sólida. Contiene protección contra clickjacking en forma de middleware X-Frame-Options, que en un navegador compatible puede evitar que un sitio se muestre dentro de un marco.
